
# 🎨 Emoji Art & Matrix Music: Manipulação de Estruturas Aninhadas

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![Data Structures](https://img.shields.io/badge/Estruturas-Dicionários%20|%20Listas%20|%20Tuplas-blueviolet?style=for-the-badge)

Este repositório contém a resolução de desafios avançados de lógica, focados na manipulação de **matrizes (listas de listas)**, processamento de **pixels (tuplas RGB)** e transformação de **dados musicais**.

## 🚀 Desafios Implementados

### 1. O Criador de Emojis (Filtro RGB)
Neste desafio, o objetivo foi processar uma grade de 5x5 pixels representada por um dicionário. 
*   **Lógica de Sombra:** Percorremos 3 níveis de loops para acessar cada canal de cor (R, G, B). Se o pixel for amarelo, aplicamos um filtro que reduz seu brilho em 50%.
*   **Conceito Chave:** Imutabilidade de Tuplas. Como não podemos alterar uma tupla diretamente, aprendemos a reconstruí-la dinamicamente durante a iteração.

### 2. Matrizes Musicais (Transposição)
Trabalhamos com a transposição de matrizes 2x2 que representam frequências sonoras.
*   **Operação:** Inversão de linhas por colunas em uma estrutura complexa de listas dentro de dicionários.
*   **Métodos Utilizados:** Uso do método `.update()` para salvar as novas matrizes transpostas na biblioteca musical.

### 3. O Integrador (Playlist de Imagens)
Um exercício de criação livre que exigiu a combinação de todos os conceitos anteriores:
*   Criação de uma "Playlist de Imagens" usando **Dicionários**, **Listas** e **Tuplas**.
*   Uso obrigatório de métodos específicos como `.pop()`, `.keys()` e `.insert()` para demonstrar domínio sobre as coleções do Python.

---

## 🧠 Conceitos de Programação Aplicados

| Estrutura | Uso no Projeto | Principais Métodos |
| :--- | :--- | :--- |
| **Dicionário** | Armazenamento de metadados (nomes, categorias). | `.items()`, `.update()`, `.keys()` |
| **Lista** | Representação de grades, matrizes e sequências. | `.append()`, `.pop()`, `.insert()` |
| **Tupla** | Definição de cores RGB (valores fixos e imutáveis). | Acesso por índice |

### Loops Aninhados (3 Níveis)
A atividade exigiu a navegação profunda em estruturas, seguindo o fluxo:
1.  **Nível 1:** Acesso ao objeto principal (Dicionário/Lote).
2.  **Nível 2:** Navegação pelas linhas ou sub-elementos (Listas).
3.  **Nível 3:** Manipulação do dado bruto (Pixel/Nota Musical).

---
